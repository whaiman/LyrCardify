## Что такое LyrCardify?

Мы верим, что сильные тексты песен заслуживают достойного визуального оформления. LyrCardify — это простой, но мощный инструмент для создания визуалов в стиле Spotify, который выходит далеко за рамки стандартных шаблонов. Если вам нужно передать настроение в Instagram Story или просто сохранить важный куплет на память — LyrCardify сделает это красиво и без лишних усилий.

## Что можно делать

- 🎵 **Магия Spotify:** Просто вставьте ссылку на трек. Мы сами подтянем обложку альбома и все детали за считанные секунды.
- 🎨 **Визуальная гармония:** Хотите, чтобы фон идеально сочетался с обложкой? Функция «Авто-цвет» подберет нужную палитру за вас.
- 🖌️ **Творческая свобода:** Выбирайте шрифты, добавляйте тени к тексту и применяйте стильные фильтры (например, дуотон, сепию или ч/б).
- 🖼️ **Формат под любые задачи:** Мгновенно переключайтесь между форматами: Квадрат (1:1), Портрет (3:4), Альбом (4:3) или Сторис (9:16).
- 📂 **Полная уникальность:** Не нравится обложка альбома? Загрузите свой собственный фон, чтобы создать что-то по-настоящему особенное.
- 💾 **Качественный экспорт:** Один клик — и ваша карточка готова в высоком разрешении PNG.

## Как запустить

Чтобы запустить LyrCardify локально:

1. **Скачайте код:**

    ```bash
    git clone https://github.com/whaiman/LyrCardify.git
    cd LyrCardify
    ```

2. **Установите зависимости:**

    ```bash
    npm install
    ```

3. **Безопасное соединение:**
    Проект использует HTTPS для корректной работы с современными API браузеров. Если у вас установлен `mkcert`, сертификаты создадутся автоматически.

4. **Запуск:**

    ```bash
    npm run dev
    ```

## Пару слов о Spotify API

Для загрузки реальных данных о треках вам понадобится свой Spotify Client ID. Его можно легко настроить один раз на странице **Настройки** прямо в приложении. Ваши ключи хранятся только в вашем браузере и никуда не передаются.

Также можно создать файл `.env` в корне:

```env
VITE_SPOTIFY_CLIENT_ID=ваш_id
VITE_SPOTIFY_CLIENT_SECRET=ваш_секрет
```

_Получить ключи можно в [Spotify Developer Dashboard](https://developer.spotify.com/dashboard/)._

## Использование

1. **Поиск**: Вставьте ссылку на трек Spotify (например, `https://open.spotify.com/track/...`).
2. **Текст**: Вставьте текст песни, который хотите выделить.
3. **Стиль**: Используйте боковую панель для настройки цветов, шрифтов и макета. Попробуйте готовые «Пресеты» для быстрого результата!
4. **Экспорт**: Нажмите кнопку «Скачать изображение», чтобы сохранить карточку.
